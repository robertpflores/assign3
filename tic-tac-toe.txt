Tic-Tac-Toe Lab Notebook

Setup
- cloned the starter repo from https://github.com/UCLA-CS-35L/assign3
- ran ./helper init inside the assign3 directory
- ran npm install — it pulled React 19, Vite, react-bootstrap, etc
- ran npm run dev, opened localhost:5173 in Chrome
- saw the default Hello World card from the starter code

Building the board
- cleared out the starter Card/Badge stuff in App.jsx
- made a Square function component, just returns a <button className="square">
- put 9 of them in the Board, grouped into 3 divs with className="board-row"
- numbered them 1–9 to make sure the grid layout worked
- grid showed up, looked right

Passing data with props
- added a value prop to Square so the board can tell each square what to show
- had to remember to use {value} inside the button, not just "value" as text
  (the tutorial warned about this but I almost did it anyway lol)
- passed value="1" through value="9" from Board to the squares

Making it interactive
- added onClick to the button in Square, wired it to a prop called onSquareClick
- set up a squares array with useState in Board, 9 nulls
- wrote handleClick(i) that copies the array with slice(), sets index i to 'X',
  then calls setSquares
- clicking a square now puts an X there

Taking turns
- added xIsNext boolean state
- if xIsNext place 'X', otherwise place 'O'
- flip xIsNext after each move
- also added a check at the top of handleClick to return early if the square
  is already filled, so you can't overwrite

Declaring a winner
- wrote calculateWinner — loops through all winning combos:
  [0,1,2], [3,4,5], [6,7,8], [0,3,6], [1,4,7], [2,5,8], [0,4,8], [2,4,6]
- if any line has 3 matching non-null values, return that value
- added winner check at the top of handleClick so clicking does nothing after
  somebody wins
- added a status div above the board: shows "Winner: X" or "Next player: O" etc

Lifting state up
- moved the squares and xIsNext state from Board up to a Game component
- Board now takes squares, xIsNext, onPlay as props
- Game has a handlePlay function that updates the squares
- everything still works the same, just organized better

Testing
- played a bunch of games manually in the browser
- checked that X goes first every time
- made sure you can't click on a square that already has a value
- tested all 8 win conditions (rows, columns, diagonals)
- confirmed no moves possible after a win
